Audit de vulnérabilités réseau — Greenbone / OpenVAS

Outil : Greenbone Security Assistant v25.3.1 · Réseau scanné : 10.48.31.0/24

Scan complet du réseau de l'Institut avec l'outil Greenbone. Plus de 1 000 résultats analysés, filtrés par score CVSS pour identifier les priorités. Chaque vulnérabilité est notée de 0 à 10 : plus le score est élevé, plus la faille est critique.

Remédiation des commutateurs réseau (Switches)

Équipements : HP 2920-48G · HPE Comware · HP ProCurve 2610

  • HP 2920-48G : Algorithmes SSH KEX faibles — non remédiable (limite firmware), documenté dans le rapport d'audit.
  • HPE Comware : Vulnérabilité SWEET32 (chiffrement DES/3DES) — remédiation via interface web d'administration.
  • HP ProCurve 2610 : Communautés SNMP par défaut corrigées via Firefox ESR 52.8.0 + plugin Java (interface d'administration legacy).

Mise à niveau des postes de travail Windows

Dell Precision Tower 5810 · Dell Precision T3610

Dell Precision Tower 5810 — Windows 10 v1909 → Windows 11

  • Résolution de l'erreur Windows Update 0x80070020 et des crashs d'explorer.exe.
  • Passage vers Windows 10 22H2 via l'outil officiel Microsoft.
  • Incompatibilités W11 contournées via FlyOOBE/Flyby11 (bypass TPM 2.0, CPU non supporté, Secure Boot).
  • Désinstallation des pilotes Dell incompatibles après la mise à niveau. ✔ Succès.

Dell Precision T3610 — Windows 7 Pro → Windows 10

  • RAID1 dégradé identifié (disque ST2000DM001 défaillant) via console Intel RST.
  • BSOD DRIVER_PNP_WATCHDOG lors de l'installation W10 — causé par le disque défaillant.
  • Clé USB bootable créée avec Rufus 4.13 en mode MBR/BIOS Legacy.
  • Installation en attente — remplacement du disque défaillant requis. Signalé au maître de stage.
